# Libft

## 📌 Descrizione

**Libft** è una libreria personale scritta in linguaggio C, sviluppata come parte del percorso formativo della scuola **42**.
Il progetto consiste nella reimplementazione di diverse funzioni della libreria standard di C, insieme ad alcune funzioni aggiuntive utili per progetti futuri.

L'obiettivo principale è comprendere a fondo il funzionamento delle funzioni di base e migliorare la gestione della memoria, delle stringhe e delle strutture dati in C.

---

## 🎯 Obiettivi del progetto

* Reimplementare funzioni della libreria standard (`libc`)
* Migliorare la comprensione della gestione della memoria
* Scrivere codice modulare e riutilizzabile
* Creare una libreria personale da usare nei progetti successivi

---

## ⚙️ Linguaggio utilizzato

* C

---

## 🛠️ Compilazione

Per compilare la libreria:

```bash
make
```

Per pulire i file oggetto:

```bash
make clean
```

Per pulire tutto:

```bash
make fclean
```

Per ricompilare:

```bash
make re
```
